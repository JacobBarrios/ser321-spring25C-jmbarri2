You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • http://18.224.33.63:9000/random?min=10&max=50 -- returns random number between the min and max number
  • http://18.224.33.63:9000/concat?str1=hello&str2=world -- returns the combined text of str1 and str2
